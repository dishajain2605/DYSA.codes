An ultra-premium, interactive personal portfolio website engineered with modern web technologies to showcase high-level expertise in **Frontend Development (React)**, **Artificial Intelligence (YOLOv8)**, and **Robotics Automation (ESP32)**.

Designed with an aesthetic inspired by top-tier creative agencies, featuring immersive 3D graphics, buttery-smooth scroll physics, and dark neon glassmorphism UI.

---

##  Elite Features
- **3D Physics Hero Canvas:** Interactive particle field and floating cyber geometry powered by `@react-three/fiber` & `Three.js`.
- **Intense GSAP Interactions:** Magnetic custom cursors, parallax scrolling algorithms, and sub-pixel stagger text reveals using the industry-standard `GSAP` animation engine.
- **Lenis Smooth Scroll Engine:** High-performance, friction-based viewport scrolling for an unparalleled luxury user experience.
- **Dynamic Neon Theme:** Custom-configured glassmorphic cards mapped to exact hexadecimal UI glowing effects (`#00f5ff`, `#8b5cf6`, `#00ff88`).
- **Responsive Architecture:** Fully tuned CSS Grid algorithms supporting seamless rendering from 280px mobile phones to 4K Ultra-Wide monitors.

## 🛠️ Technology Stack
* **Framework:** React + Vite
* **Styling:** Tailwind CSS v4 + Pure Vanilla CSS 
* **Animations:** GSAP, ScrollTrigger, `react-type-animation`
* **3D & WebGL:** Three.js, React Three Fiber, Drei
* **Motion Physics:** Lenis Scroll, GSAP QuickTo
* **Icons:** React Icons

##  Quick Start (Local Setup)

To experience the platform locally:

```bash
# Clone the repository
git clone https://github.com/dishajain2605/DYSA.codes.git

# Navigate to the project directory
cd DYSA.codes

# Install dependencies (React, GSAP, ThreeJS, Lenis)
npm install

# Start the Vite development server locally
npm run dev
```

Visit `http://localhost:5173` to view the live dashboard!
